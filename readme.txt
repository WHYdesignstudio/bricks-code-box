Ein leichtgewichtiges Bricks-Addon, das ein Code-Box-Element mit Prism.js, Copy-Button, Zeilennummern und optionalen Labels hinzufügt.

== Description ==

Fügt dem Bricks Builder ein leistungsfähiges Code-Box-Element hinzu – mit Syntax-Highlighting (Prism.js), Copy-Button, Zeilennummern, Dark/Light-Themes, optionalem Dateinamen und mehr.

== Installation ==
1. ZIP hochladen und aktivieren (Plugins → Installieren → Hochladen)
2. In Bricks ein neues „Code Box“-Element hinzufügen
